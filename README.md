# CryptoChain

Simple react web to display cryptocurrencies prices in realtime.
* FrontEnd : React.js
* Backend : Node.js/Javascript
* API used : * [CoinGecko](https://www.coingecko.com/en)

## Installation

```sh
git clone https://github.com/Abdellatif-E/crypto.git
```

```sh
npm install
```
